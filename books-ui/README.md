# Books UI

Angular frontend for managing records in the bookstore `books` table.

## API usage

- `GET http://localhost:8081/api/books/metadata/columns`
- `GET http://localhost:8081/api/books`
- `POST http://localhost:8081/api/books`
- `PUT http://localhost:8081/api/books/{id}`
- `DELETE http://localhost:8081/api/books/{id}`

## Run

Start the Spring Boot `catalog-service` first, then run:

```bash
npm start
```

Open `http://localhost:4200`.

## Build

```bash
npm run build
```
